让git push命令不再需要密码

2017-02-09
  • Git
  • GitHub

最近利用jekyll写博客，为的就是博客管理方便，但是在上传博客的时候使用git push命令每次都得输入github帐号和密码特别的不方便，于是就搜了一下。

在这篇文章里提到，GitHub获得远程库时，有ssh方式和https方式。
